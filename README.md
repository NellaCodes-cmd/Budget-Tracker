# Personal Budget Tracker

A clean, responsive budget tracking web application built with vanilla HTML, CSS and JavaScript.

## Features
- ✅ Add income and expense transactions
- ✅ Real-time balance calculation
- ✅ Color coded transactions (green/red)
- ✅ Delete transactions
- ✅ Separated HTML, CSS and JavaScript files

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)

## How to Run
1. Clone the repository
2. Open `index.html` in any browser
